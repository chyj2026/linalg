Lead

Individual afternoon session re-deriving the 2×2 conic diagonalization as M·[ε₁ ε₂] = [ε₁ ε₂]·Λ, then defining the eigenvalue problem (M-λI)v = 0 and showing the resulting quadratic recovers the same λ₁+λ₂ = a+c and λ₁λ₂ = ac-b² invariants.

Key theorems and identities (from the session)

  • Eigenvalues defined via M·v = λv as fixed/dilated subspace
  • (M-λI)v = 0 requires det(M-λI) = 0 for nontrivial solutions
  • Vieta on characteristic polynomial gives trace and det
